Discord サーバーのロール管理を簡単に

招待リンクから自動でロールを付与するDiscord Botシステム

セットアップ方法 GitHub

主な機能

🔗

招待リンク生成

特定のロールが自動で付与される招待リンクを簡単に生成できます

⏰

有効期限設定

招待リンクに有効期限を設定して、セキュリティを保つことができます

📊

使用回数制限

招待リンクの使用回数を制限して、適切な人数管理を行えます

👑

管理者機能

サーバー管理者は全ての招待リンクを一覧表示・削除できます

👤

個人管理

ユーザーは自分が作成した招待リンクを管理できます

🌐

Webインターフェース

美しいWebページでDiscordサーバーへの参加体験を提供します

利用可能なコマンド

/generate_invite_link

ロール招待リンクを生成します（管理者専用）

  • 対象ロールの選択
  • 有効期限の設定
  • 使用回数制限の設定

/list_server_invite_links

サーバーの全招待リンクを一覧表示・削除（管理者専用）

  • 全てのリンクの表示
  • 有効性の確認
  • 一括削除機能

/list_my_invite_links

自分が作成した招待リンクを一覧表示・削除

  • 個人のリンク管理
  • 使用状況の確認
  • 不要なリンクの削除

セットアップ方法

1

前提条件

  • Python 3.8以上
  • PostgreSQLデータベース
  • Discord Bot アプリケーション
  • Discord OAuth2 アプリケーション
2

リポジトリのクローン

git clone https://github.com/tejastice/invite-and-role-bot.git
cd invite-and-role-bot
3

環境変数の設定

各フォルダの .env.example を参考に .env ファイルを作成してください。

4

Discord Botの起動

cd discord_bot
pip install -r requirements.txt
python bot.py
5

Webアプリケーションの起動

cd get_role
pip install -r requirements.txt
python app.py
